Atlas Mobile — Q3 review

We've shipped the desktop client to a small private beta. Feedback on collaboration is positive but reviewers consistently ask for a way to work from their phone.

What we know

  • 12 of 18 beta reviewers commented from a phone in the past two weeks by replying via Drive's web UI — the desktop sidecar picked those up correctly within ~10 seconds.
  • Drive comments arrive in the desktop sidecar within 5–10 seconds while the window is focused.
  • The sidecar format is portable; nothing in the v2 envelope is desktop-specific.

What's new this milestone

  1. Standalone APK distribution (out of the Play Store for v1)
  2. Open via the Storage Access Framework — Drive, Dropbox, OneDrive, local
