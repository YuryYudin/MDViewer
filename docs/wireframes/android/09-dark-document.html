Atlas Mobile — Q3 review

We've shipped the desktop client to a small private beta. Feedback on collaboration is positive but reviewers consistently ask for a way to work from their phone.

What we know

  • 12 of 18 beta reviewers commented from a phone in the past two weeks by replying via Drive's web UI — the desktop sidecar picked those up correctly within ~10 seconds.
  • Drive comments arrive in the desktop sidecar within 5–10 seconds while the window is focused.
  • The sidecar format is portable; nothing in the v2 envelope is desktop-specific. A reviewer can carry a folder between devices unchanged.

What's new this milestone

  1. Standalone APK distribution (out of the Play Store for v1)
  2. Open via the Storage Access Framework — Drive, Dropbox, OneDrive, local — all show up in the picker
  3. Comment-only edits — no Markdown editing yet

Anchor model

Anchors are W3C TextQuoteSelectors flattened: exact + prefix + suffix + offsets. The mobile client reuses the desktop's Bitap fuzzy resolver via the shared Rust core.

The unit of collaboration is a folder. Comments live next to the document. The mobile client honors that contract — sidecars persist as DocumentFile siblings.
