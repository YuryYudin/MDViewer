MDViewer for Android — wireframes

Ten frames cover the v1 user journey. Each links to its standalone HTML so you can resize, inspect, and review interaction details. The phone frame is sized at 412 × 891 dp (Pixel 7 / 8 family). Material 3 light and dark theme tokens are scoped under .m3-light / .m3-dark.
