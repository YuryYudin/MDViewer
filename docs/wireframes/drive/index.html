Google Drive integration — wireframes

Eight screens covering the new surfaces. Click into any one. The "Back" link in each screen returns here.

01 · Settings — Drive (Not connected) First-run state. "Connect to Drive" button + advanced BYO toggle. 02 · Settings — Drive (Connected) Account chip, backend mode, custom OAuth fields, [Disconnect]. 03 · Connect flow Click Connect → loopback browser opens → success state. 04 · Open from Drive (URL paste) File menu's new entry. URL paste with validation states. 05 · Document with Drive comments Status pill, collaborators chip, sidebar with Drive-sourced threads. 06 · Offline state Banner + queued comments waiting to sync. 07 · Save conflict (Drive doc) Doc was modified on Drive while you were editing. 08 · Drive-detected toast OS picker opened a Drive Desktop file but Drive isn't connected yet.
