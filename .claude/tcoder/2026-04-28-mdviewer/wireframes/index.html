MDViewer wireframes

Cross-platform Tauri 2 markdown viewer with collaborative comments. Click any screen below to view its layout. Wireframes show structure and interaction surface only — visual polish comes later.

01 · Startup & recent files
First window after launch with no doc open. Recent files list, Open / New buttons.
02 · First-launch profile setup
Modal asking for display name and an avatar color. Stored locally.
03 · Rendered document, no comments
A doc open in a tab, rendered, sidebar shows the empty-comments state.
04 · Selection & comment popover
User has selected text in the rendered view; "Comment" popover floats above it.
05 · Document with active comment threads
Several anchored highlights, threads in the sidebar, one thread active.
06 · Thread with replies and resolve
A focused thread showing replies, reply composer, resolve button.
07 · Edit mode (raw markdown)
Mode switched to Edit; raw .md textarea; comments visible but greyed in sidebar.
08 · Text-conflict resolution
Diff view shown when an incoming .md disagrees with the local copy.
09 · Orphaned comments
Anchors that fuzzy-reattachment couldn't relocate after edits.
10 · Share document & comments
Dialog explaining the .md + sidecar pair being prepared for sending.
11 · Settings (profile + appearance)
Edit display name, color, theme, and other prefs after first launch.
12 · Dark mode example
Document-with-comments view rendered in dark theme.
