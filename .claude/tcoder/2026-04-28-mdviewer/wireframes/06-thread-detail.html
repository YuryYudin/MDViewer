RFC-031: Per-tenant Rate Limiting

Summary

Today the gateway enforces a single global request budget. As we onboard larger customers, a single tenant's traffic spike can degrade response times for every other tenant. This RFC proposes a per-tenant token-bucket limiter at the gateway.

Design

Each gateway pod owns a local token bucket per (tenant, route-class) pair, replenished from a global ledger every refresh_interval seconds.
