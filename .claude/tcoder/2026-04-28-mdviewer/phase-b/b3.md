# B3: save_document IPC and watcher self-write suppression

**Avoid:**
- Do NOT write directly with `std::fs::write` and hope for atomicity. WHY: a crash between truncate and write produces a zero-byte `.md`, destroying the user's document. Write to `<path>.tmp`, fsync, then `rename` — the same primitive every editor uses for crash safety.
- Do NOT register the self-write *after* writing. WHY: notify's event delivery is fast enough that the watcher can fire before `save_document` returns. Register the `(path, content_hash)` entry *before* the rename so the suppression list is primed when the event arrives.
- Do NOT collapse the suppression key to `path` alone. WHY: an external editor could write the same file at almost the same time; matching only on path would silently drop their event. The design specifies the tuple `(path, content_hash, timestamp)` — keep all three.
- Do NOT treat a sidecar save as a markdown save. WHY: B2 distinguishes `WatchedKind::Markdown` and `WatchedKind::Sidecar`; reusing the same `save_document` for sidecars would route the suppression entry into the wrong bucket and the sidecar would still trigger a phantom auto-merge prompt.

use mdviewer_lib::settings::ExternalChangeBehavior;
use mdviewer_lib::watcher::{ExternalChange, Watcher};
use std::fs;
use std::time::Duration;
use tempfile::TempDir;

#[test]
fn save_document_does_not_trigger_reload() {
    let tmp = TempDir::new().unwrap();
    let md = tmp.path().join("doc.md");
    fs::write(&md, "v1").unwrap();

    let (tx, rx) = std::sync::mpsc::channel();
    let mut w = Watcher::new(tx).unwrap();
    w.set_external_change_behavior(ExternalChangeBehavior::Reload);
    w.watch_md(&md).unwrap();

    let r = save_document(&md, b"v2").unwrap();
    w.record_self_write(&md, r.content_hash);

    // No event should reach us within the suppression window.
    assert!(rx.recv_timeout(Duration::from_millis(500)).is_err());

fn external_write_after_save_still_triggers() {
    let tmp = TempDir::new().unwrap();
    let md = tmp.path().join("doc.md");
    fs::write(&md, "v1").unwrap();

### Step 2: Implement save_document

Modify `src-tauri/src/document.rs` — append:

```rust
use crate::watcher::quick_hash;
use std::path::Path;

pub struct SaveResult {
    pub bytes_written: usize,
    pub content_hash: u64,
}

pub fn save_document(path: &Path, contents: &[u8]) -> std::io::Result<SaveResult> {
    use std::io::Write;
    let tmp = path.with_extension(
        path.extension().and_then(|s| s.to_str())
            .map(|e| format!("{e}.tmp")).unwrap_or_else(|| "tmp".into())
    );
    {
        let mut f = std::fs::File::create(&tmp)?;
        f.write_all(contents)?;
        f.sync_all()?;
    }
    std::fs::rename(&tmp, path)?;
    Ok(SaveResult { bytes_written: contents.len(), content_hash: quick_hash(contents) })
}
```

### Step 3: Wire IPC + watcher

Modify `src-tauri/src/main.rs` — register a `save_document` IPC command. Inside the handler:

```rust
#[tauri::command]
fn save_document(state: State<'_, Ws>, watcher: State<'_, Mutex<Watcher>>, path: PathBuf, contents: String) -> Result<(), String> {
    let r = mdviewer_lib::document::save_document(&path, contents.as_bytes()).map_err(|e| e.to_string())?;
    watcher.lock().unwrap().record_self_write(&path, r.content_hash);
    // Update the in-memory tab source/render via Workspace.
    state.lock().map_err(|e| e.to_string())?.refresh_tab(&path).map_err(|e| e.to_string())?;
    Ok(())
}
```

Add `Workspace::refresh_tab(path)` to `src-tauri/src/workspace.rs` — re-reads the file, re-renders via `document::render_markdown`, and replaces the cached `RenderResult`/`source` in the tab. Concrete shape:

```rust
// in src-tauri/src/workspace.rs (B3 modifies)
pub fn refresh_tab(&mut self, path: &Path) -> Result<()> {
    let canonical = path.canonicalize().unwrap_or_else(|_| path.to_path_buf());
    let source = std::fs::read_to_string(&canonical)
        .with_context(|| format!("read {:?}", canonical))?;
    let s = self.settings.get();
    let opts = RenderOptions {
        syntax_highlighting: s.editor.syntax_highlighting,
        mermaid_enabled: s.editor.mermaid_enabled,
    };
    let render = render_markdown(&source, &opts);
    let tab = self.tabs.values_mut()
        .find(|t| t.path == canonical)
        .ok_or_else(|| anyhow::anyhow!("no open tab for path {:?}", canonical))?;
    tab.source = source.clone();
    tab.render = render;
    tab.last_saved_snapshot = Some(source);
    Ok(())
}
```

(B2 already declares `record_self_write` as `pub`, so no visibility change is needed in `watcher.rs`; the integration test at `src-tauri/tests/watcher.rs` calls it directly.)
