# B2: File watcher with external-change behavior

**Avoid:**
- Do NOT spawn a watcher per `Workspace::open_document` call. WHY: `notify` opens a kqueue / inotify / FSEvents handle per recursion root, and a many-tabs session would exhaust file descriptors. Use a single watcher with `watcher.watch(path, NonRecursive)` calls per open file.
- Do NOT debounce events here. WHY: B3 needs the raw event stream so its self-write suppression can match by exact `(path, hash, timestamp)`. Debouncing first would coalesce a save followed by an external write into one event and obscure the source.
- Do NOT consult settings synchronously inside the notify callback. WHY: notify's worker thread cannot block on a Tokio mutex. Snapshot the relevant `external_change_behavior` and `auto_merge` fields into the watcher state via the settings change-event channel and read them lock-free in the callback.
- Do NOT collapse `.md` and sidecar events. WHY: the design routes `.md` external changes into `document.rs` and sidecar external changes into `sidecar.rs` with the auto-merge policy. A single "doc changed" event would force the consumer to re-derive which path it is and risk applying the wrong policy.

fn waitfor<T>(rx: &std::sync::mpsc::Receiver<T>) -> T {
    rx.recv_timeout(Duration::from_secs(2)).expect("event")

### Step 3: Implement watcher.rs

Create `src-tauri/src/watcher.rs`:

```rust
use crate::settings::ExternalChangeBehavior;
use anyhow::Result;
use notify::{Event, EventKind, RecommendedWatcher, RecursiveMode, Watcher as NotifyWatcher};
use std::collections::{HashMap, HashSet};
use std::path::{Path, PathBuf};
use std::sync::{Arc, Mutex};
use std::sync::mpsc::Sender;

#[derive(Debug, Clone, Copy, PartialEq, Eq)]
pub enum WatchedKind { Markdown, Sidecar }

#[derive(Debug, Clone, Copy, PartialEq, Eq)]
pub enum ExternalChange { Ask, Reload, Ignore }

#[derive(Debug, Clone)]
pub struct ExternalChangeEvent {
    pub path: PathBuf,
    pub kind: WatchedKind,
    pub action: ExternalChange,
}

#[derive(Default)]
struct State {
    behavior: Option<ExternalChangeBehavior>,
    md_paths: HashSet<PathBuf>,
    sidecar_paths: HashSet<PathBuf>,
    unsaved: HashMap<PathBuf, bool>,
    /// Self-write suppression list — entries are (path, content_hash) within a
    /// short timestamp window. B3 populates this on save_document; this module
    /// only checks against it.
    self_writes: Vec<SelfWrite>,
}

#[derive(Debug, Clone)]
pub(crate) struct SelfWrite {
    pub path: PathBuf,
    pub content_hash: u64,
    pub at: std::time::Instant,
}

pub struct Watcher {
    inner: RecommendedWatcher,
    state: Arc<Mutex<State>>,
}

impl Watcher {
    pub fn new(out: Sender<ExternalChangeEvent>) -> Result<Self> {
        let state = Arc::new(Mutex::new(State::default()));
        let st = state.clone();
        let watcher = RecommendedWatcher::new(
            move |res: notify::Result<Event>| {
                let Ok(event) = res else { return; };
                if !matches!(event.kind, EventKind::Modify(_) | EventKind::Create(_)) { return; }
                let mut s = st.lock().unwrap();
                for path in event.paths {
                    let kind = if s.md_paths.contains(&path) { WatchedKind::Markdown }
                        else if s.sidecar_paths.contains(&path) { WatchedKind::Sidecar }
                        else { continue };

                    // Self-write suppression
                    if let Ok(bytes) = std::fs::read(&path) {
                        let hash = quick_hash(&bytes);
                        if s.self_writes.iter().any(|sw| sw.path == path && sw.content_hash == hash
                            && sw.at.elapsed() < std::time::Duration::from_secs(2)) {
                            continue;
                        }
                    }

                    let action = if *s.unsaved.get(&path).unwrap_or(&false) {
                        ExternalChange::Ask
                    } else {
                        match s.behavior {
                            Some(ExternalChangeBehavior::Ask) | None => ExternalChange::Ask,
                            Some(ExternalChangeBehavior::Reload) => ExternalChange::Reload,
                            Some(ExternalChangeBehavior::Ignore) => continue,
                        }
                    };
                    let _ = out.send(ExternalChangeEvent { path: path.clone(), kind, action });
                }
            },
            notify::Config::default(),
        )?;
        Ok(Self { inner: watcher, state })
    }

    pub fn set_external_change_behavior(&mut self, b: ExternalChangeBehavior) {
        self.state.lock().unwrap().behavior = Some(b);
    }

    pub fn watch_md(&mut self, p: &Path) -> Result<()> {
        self.inner.watch(p, RecursiveMode::NonRecursive)?;
        self.state.lock().unwrap().md_paths.insert(p.to_path_buf());
        Ok(())
    }

    pub fn watch_sidecar(&mut self, p: &Path) -> Result<()> {
        self.inner.watch(p, RecursiveMode::NonRecursive)?;
        self.state.lock().unwrap().sidecar_paths.insert(p.to_path_buf());
        Ok(())
    }

    pub fn mark_unsaved(&self, p: &Path, dirty: bool) {
        self.state.lock().unwrap().unsaved.insert(p.to_path_buf(), dirty);
    }

    /// `pub` (not `pub(crate)`) so integration tests under `src-tauri/tests/`
    /// — which live in their own crate — can drive the suppression list
    /// directly. B3's tests in `tests/watcher.rs` rely on this visibility.
    pub fn record_self_write(&self, p: &Path, hash: u64) {
        let mut s = self.state.lock().unwrap();
        s.self_writes.push(SelfWrite { path: p.to_path_buf(), content_hash: hash, at: std::time::Instant::now() });
        // Trim old entries
        s.self_writes.retain(|sw| sw.at.elapsed() < std::time::Duration::from_secs(10));
    }
}

pub fn quick_hash(bytes: &[u8]) -> u64 {
    use std::hash::{Hash, Hasher};
    let mut h = std::collections::hash_map::DefaultHasher::new();
    bytes.hash(&mut h);
    h.finish()
}
```

Both `record_self_write` and `quick_hash` are public; they cross the integration-test boundary in B3. Update b3.md's "expose `record_self_write` publicly" note to note that no visibility change is needed — it's already `pub` here.

### Step 4: Wire main.rs

Modify `src-tauri/src/main.rs`'s `setup` closure to construct a `Watcher`, register it as Tauri-managed state alongside the existing `Mutex<Workspace>`, and forward events:

```rust
.setup(|app| {
    // ... existing Workspace setup ...
    let app_handle = app.handle().clone();
    let (tx, rx) = std::sync::mpsc::channel::<mdviewer_lib::watcher::ExternalChangeEvent>();
    let watcher = mdviewer_lib::watcher::Watcher::new(tx)?;
    app.manage(std::sync::Mutex::new(watcher));

    // Forward watcher events to the frontend.
    std::thread::spawn(move || {
        for ev in rx {
            let _ = app_handle.emit("external-change", &ev);
        }
    });

    // Subscribe to settings changes and re-apply external-change behaviour.
    // ... per the existing settings-subscribe pattern ...
    Ok(())
})
```

The explicit `app.manage(std::sync::Mutex::new(watcher))` line is what B3's `save_document` IPC handler relies on to find `State<'_, Mutex<Watcher>>`. Without it B3 fails at runtime with "no managed state of type Mutex<Watcher>". The `emit` call assumes `tauri::Emitter` is in scope — add `use tauri::Emitter;` to the imports.

Modify `src-tauri/src/lib.rs` — add `pub mod watcher;`.
